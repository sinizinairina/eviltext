EvilText

Websites for you and your projects.

Transform plain text into blog, wiki or website.

Quick Start

1

Install Node.js

Follow instruction on the official Node.js site.

2

Install EvilText

npm install -p eviltext

3

Create sample site

Clone sample with Git or download it.

git clone http://github.com/sinizinairina/eviltext-blog-example

4

Build and see it live

cd eviltext-blog-example

eviltext build

eviltext serve

Open http://localhost:3000 in your Browser
